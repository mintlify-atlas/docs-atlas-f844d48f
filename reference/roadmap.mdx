This roadmap outlines planned features and improvements for future releases of PFP Checker. Features are subject to change based on community feedback and development priorities.

<Info>
  Have a feature request? Open an issue on [GitHub](https://github.com/j4ytr1n1ty/pfp-checker/issues) to share your ideas.
</Info>

## Upcoming Features

### v0.5.0

<AccordionGroup>
  <Accordion title="Server-specific commands and settings" icon="server">
    **Status:** Planned
    
    Add support for customizing bot behavior on a per-server basis, including:
    - Server-specific monitoring preferences
    - Custom command prefixes
    - Configurable notification channels
  </Accordion>

  <Accordion title="Role-based permissions for commands" icon="shield">
    **Status:** Planned
    
    Implement granular permission controls:
    - Allow server admins to restrict command usage by role
    - Create custom permission tiers
    - Set up command-specific access controls
  </Accordion>

  <Accordion title="Track emojis and server banner changes" icon="image">
    **Status:** Planned
    
    Expand tracking capabilities beyond icons:
    - Monitor custom emoji additions and removals
    - Track server banner changes
    - Archive emoji and banner history
  </Accordion>

  <Accordion title="Improved error handling and logging" icon="bug">
    **Status:** Planned
    
    Enhance diagnostic capabilities:
    - More detailed error messages
    - Better logging for troubleshooting
    - Improved error recovery mechanisms
  </Accordion>
</AccordionGroup>

---

### v0.6.0

<AccordionGroup>
  <Accordion title="Web dashboard for viewing statistics" icon="chart-line">
    **Status:** Planned
    
    Create a web-based dashboard:
    - View tracking statistics in a browser
    - Interactive charts and graphs
    - Export data in various formats
    - Public profile pages for users
  </Accordion>

  <Accordion title="Support for backing up images to different providers" icon="cloud">
    **Status:** Planned
    
    Add support for multiple image hosting services:
    - Configure alternative image storage providers
    - Automatic failover between providers
    - Self-hosted image storage options
  </Accordion>

  <Accordion title="Generate GIF/video of profile picture changes" icon="film">
    **Status:** Planned
    
    Create visual timelines:
    - Generate animated GIFs showing all profile picture changes
    - Export video compilations
    - Customize playback speed and transitions
  </Accordion>

  <Accordion title="Track status changes" icon="circle-dot">
    **Status:** Planned
    
    Monitor Discord status updates:
    - Track custom status changes
    - Monitor activity status (online, idle, DND)
    - View status history
  </Accordion>
</AccordionGroup>

## Long-term Goals

These are vision items without specific version targets:

<CardGroup cols={2}>
  <Card title="Multiple Database Backends" icon="database">
    Support for PostgreSQL and MySQL in addition to SQLite for better scalability and production deployments.
  </Card>

  <Card title="Third-party API" icon="plug">
    RESTful API for third-party integrations, allowing developers to build custom tools and services.
  </Card>

  <Card title="User Behavior Analytics" icon="chart-mixed">
    Analytics dashboard showing patterns in profile picture changes, peak change times, and trending styles.
  </Card>

  <Card title="ML-Powered Detection" icon="brain">
    Machine learning to detect related profile pictures, identify similar images, and suggest connections.
  </Card>
</CardGroup>
